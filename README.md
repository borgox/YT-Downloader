# YT-Downloader

## Description
This is a tool that allows you to download any youtube video by its URL.

## Instructions
- Download ZIP file
- Extract it
- Open the folder YT-Downloader/WindowsUtils
- Run install.bat
- Run start.bat

## Note
- For the web version you have to do the same process as befor.
